Multi-platform general-purpose environment for business process simulation modeling

Business processes is a broad concept that includes in particular^
trading,
automation of idustrial production (SCADA).
smart home,
computer games.

For them, it is possible to use general mathematics, defined by mnemonic schemes. This project is dedicated to such an environment. Like the Unity gaming environment, this development is multiplatform. The proposed environment has a clean architecture and supports the following programming languages:
C#,
TypeScript,
Java.

The following languages are planned to be added:

C++,
Python,
Go,
Rust.

The environment also includes computer graphics. It currently supports the following 3D file formats:

.dae (Collada),
.obj,
.ac,
.xaml

 	It is planned to add WebGL support for display in the browser. Support for user interests is also planned.
This development was used to solve the following problems.
Trading analysts (Interbroker).
Manufacturing control systems (SCADA),
Smart home control systems,
Determining the weight of  pigs using a 3D camera (Piggy Check).
